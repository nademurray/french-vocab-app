# French Vocab App — LLM Wiki

A maintained knowledge base for the French Vocab PWA project. Update these pages at the start/end of each session. New Claude sessions should read this index first, then pull relevant pages.

## Pages

| Page | What it covers |
|---|---|
| [project.md](project.md) | Purpose, user context, goals, constraints |
| [architecture.md](architecture.md) | Tech stack, file structure, data model, key design decisions |
| [sessions.md](sessions.md) | Chronological log of what was built/changed each session |
| [issues.md](issues.md) | Bugs encountered and how they were resolved |
| [backlog.md](backlog.md) | Known limitations, planned improvements, ideas |

## Schema / conventions

- Dates: YYYY-MM-DD
- Each session log entry starts with the date and a one-line summary
- Issues are logged even after resolution — the fix and root cause matter for future sessions
- Backlog items tagged: `[V2]` = planned next version, `[idea]` = not committed, `[blocked]` = waiting on something
