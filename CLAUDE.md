# French Vocab App

## Start of every session

1. Read `wiki/index.md` — table of contents and conventions
2. Read `wiki/sessions.md` — bottom entry = current state of the project
3. Pull any other wiki pages relevant to the task at hand:
   - `wiki/architecture.md` — before touching any code
   - `wiki/issues.md` — before debugging
   - `wiki/backlog.md` — before adding features

## End of every session

Update the wiki before closing:
- Add an entry to `wiki/sessions.md` (date, summary, what changed, state at end)
- Update `wiki/issues.md` if any bugs were hit or fixed
- Update `wiki/backlog.md` if new ideas or limitations surfaced
- Commit and push all wiki changes
